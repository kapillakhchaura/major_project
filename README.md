AI Smart Medical Report System

AI Smart Medical Report System is an AI-powered healthcare web application developed using Python, Streamlit, and Machine Learning. The system predicts diseases like Diabetes, Heart Disease, and Kidney Disease using medical parameters and symptom analysis. It also provides risk analysis, PDF medical report generation, OCR-based report scanning, secure authentication, and an interactive healthcare dashboard for smart healthcare assistance.

Features
Disease Prediction using Machine Learning
Symptom-based AI Analysis
Risk Score & Risk Category Detection
PDF Medical Report Generation
Upload & Analyze Medical Reports
OCR Support for Image Reports
Secure Login & Signup System
User & Admin Dashboard
Interactive Streamlit UI
